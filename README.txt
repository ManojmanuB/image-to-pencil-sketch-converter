Pro Pencil Sketch App

1. Extract the pencil_sketch_app.zip to any folder.

2. Open Command Prompt in that folder:
   cd path\to\pencil_sketch_app

3. (Optional but recommended) Create virtual environment:
   python -m venv venv
   venv\Scripts\activate

4. Install required packages:
   pip install -r requirements.txt

5. Run the app:
   python -m streamlit run app.py

6. Use the tabs to upload an image or use your webcam.
   Adjust sliders and styles from the sidebar.
   Download sketches using the download buttons.

Enjoy your professional pencil sketch app!
